Python 

Pandas 

Matplotlib 

Jupyter Notebook / Any IDE 

# Features
Contains information from the Roman Empire using the csv from multiple sources to see the progression of the Roman Empire as well as the downfall of it throughout its entire lifespan. The project shows the correlation of wheat prices and its correlation to the economy, current ruling emperors, as well as the currency used at the time. It also display a regression line and a polynomial regression curve 
to illustrate the correlation and the downward trend of the empire.

# Results
 * Analysis of the Roman Empire
 * Trends between Economy and Political tensions
 * How successful an Emperors reign was due to if death cause was by assassination/execution or by other means
 * Many unknown certainties as the analysis is from an Empire that exists more than a millennium ago
 * Future Implementation: influences of Rome throughout other provinces
